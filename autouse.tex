\linksubsection{autouse}{Autouse Fixtures}

If you want to prepare the environment for all tests implicitly, use \py{autouse=True}.
This runs the fixture for all tests for which it is visible (e.g. in the same module or the same class, depending on definition location):

\begin{minted}[autogobble,highlightlines=1]{python}
    @pytest.fixture(autouse=True)
    def tmp_homedir(tmp_path, monkeypatch):
        monkeypatch.setenv("HOME", str(tmp_path))
        return tmp_path

    def test_a(tmp_homedir):
        assert not list(tmp_homedir.iterdir())

    def test_b():
        home = pathlib.Path.home()
        assert not list(home.iterdir())
\end{minted}

\linksubsection{usefixtures}{usefixtures marker}
Using fixtures without specifying them as function args:

\begin{minted}[autogobble, highlightlines=6]{python}
    @pytest.fixture
    def tmp_homedir(tmp_path, monkeypatch):
        monkeypatch.setenv("HOME", str(tmp_path))
        return tmp_path

    @pytest.mark.usefixtures("tmp_homedir")
    class TestHomeDir:
        def test_empty(self):
            home = pathlib.Path.home()
            assert not list(home.iterdir())
\end{minted}
